# Permissions

| Permission | Default | Use |
|---|---|---|
| `openeco.command.balance` | true | Check own balance |
| `openeco.command.baltop` | true | View leaderboard |
| `openeco.command.pay` | true | Send money |
| `openeco.command.history` | true | View own history |
| `openeco.command.balance.others` | op | Check another player's balance |
| `openeco.command.history.others` | op | View another player's history |
| `openeco.command.eco.give` | op | Give money |
| `openeco.command.eco.take` | op | Take money |
| `openeco.command.eco.set` | op | Set balance |
| `openeco.command.eco.reset` | op | Reset balance |
| `openeco.command.eco.delete` | op | Delete an account and that account's history |
| `openeco.command.eco.freeze` | op | Freeze an account |
| `openeco.command.eco.unfreeze` | op | Unfreeze an account |
| `openeco.command.eco.rename` | op | Rename an account's display name |
| `openeco.command.eco.reload` | op | Reload config and messages |
| `openeco.admin` | op | Grants all admin permissions above |
